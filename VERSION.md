# VERSION

## 0.7.5 - 2026-03-28

- 업로드 파일 읽기 경로를 `path.resolve` 기반으로 검증해 `storage/uploads` 바깥 파일 접근(경로 순회)을 차단

- 표지, 기사 지면, Contributors/Colophon 페이지를 동일한 HTML/CSS 지면 템플릿으로 렌더링하도록 재구성
- `Math-Zine` 로고를 PDF 표지/기사 상단/Colophon에서 블랙레터 폰트 스택으로 통일
- 로컬 업로드 이미지를 PDF 생성 시 data URI로 인라인 변환해 플레이스홀더 출력 문제를 줄임
- 코드 블록, 수식 블록, 굵게/기울임/링크 등 인라인 서식을 브라우저 렌더 기준으로 출력하도록 조정
- 인쇄 미리보기용 `/issues/[issueSlug]/print` 페이지를 추가
- `pdfkit` 의존성을 제거하고 README를 새 PDF 파이프라인 기준으로 갱신
